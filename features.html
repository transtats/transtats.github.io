Translation Stats

Track translation progress

In development of localised software, translation progress needs to be monitored on release-to-release basis. Transtats collects template files, analyse their translation statistics and generate graphs.

Custom graph rules

How about consolidate stats for the list of packages for a language set? Graph based on rules facilitate representation of stats for more specific requirements for better results.

Compare with release streams

Sometimes translation platform shows good progress yet it is found that package released does not contain the latest. Transtats never let you down at translation check point.

Frequent sync with repositories

Transtats pulls data from code and translation repositories at regular intervals for latest data representations. Snapshots are also kept to manage translation growth analysis.
